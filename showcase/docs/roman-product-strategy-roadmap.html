Repo Recommendations

SQL Playground

Primary recommendation: Guided Challenge Mode

  • Add challenges, hints, expected result validation, and completion tracking.
  • Best feature for proving product design on top of SQL and schema tooling.
  • Strongest single flagship for internship portfolio impact.

Secondary recommendation: Shareable query links and history.

Jessica Career OS

Primary recommendation: Follow-Up Automation

  • Add reminder logic, stage-based nudges, stale application flags, and prewritten follow-up copy.
  • Best feature for proving workflow and business-software thinking.
  • Creates a stronger end-to-end story than job discovery alone.

Secondary recommendation: Recruiter and hiring-manager CRM layer.

Roman Command Center

Primary recommendation: Weekly Review

  • Add trend summaries, completed-vs-planned analysis, and next-week focus suggestions.
  • Best feature for proving analytics thinking and dashboard maturity.
  • High-value feature without needing a major architecture change.

Secondary recommendation: CSV export and import.

Physics Study Bot

Primary recommendation: Adaptive Study Loop

  • Add question flow, answer review, weak-topic tracking, and study recommendations.
  • Best path from concept repo to real product.
  • Useful if the goal is more breadth across product categories.
